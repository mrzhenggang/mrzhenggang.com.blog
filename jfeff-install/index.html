上传，增加可执行权限，执行安装

在windows下下载后的文件名为 jfeff-9.7.1-linux-x64-installer.man ，使用 ftp 工具上传到服务器某目录。然后使用 chmod +x 文件名 命令增加可执行权限。使用 ./文件名 来执行安装命令，如下图：

  1. 修改打开脚本 jfeff.command 的路径为自己的目录；
  2. 为命令添加软连接；
  3. 写入 ~/.bashrc 文件可以自动加载环境。
